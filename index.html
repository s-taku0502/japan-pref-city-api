Japan Pref-City API

都道府県名から市区町村の一覧を簡単に取得できる、軽量でシンプルなREST API

APIの概要

最新の行政区画情報（CSV）を元に更新された市区町村データを提供します。 CORS対応済みで、フロントエンドからの直接呼び出しが可能です。

GET /api/prefectures
GET /api/municipalities?prefecture={都道府県名}

データ統計

-
都道府県数
-
総市区町村数

データ・エクスプローラー
