# Japan Pref-City API

都道府県名から市区町村の一覧を簡単に取得できる、軽量でシンプルなREST APIです。

このプロジェクトは、日本全国の自治体データを活用したい開発者や、プログラミング学習者向けに公開されています。

---

## 🚀 特長

- **シンプル設計**: 外部データベースを必要とせず、静的なJSONファイルのみで動作します。
- **高速レスポンス**: メモリ上のデータを返すため、非常に低レイテンシです。
- **データ更新済み**: 添付された最新のCSVデータに基づき、全国の市区町村リストが更新されています。
- **CORS対応**: デフォルトでCORSが有効になっており、フロントエンドからの直接呼び出しが可能です。

---

## 📖 はじめに

このリポジトリを参照、または利用される方への説明です。

### データの出典と更新について
本APIで使用している市区町村データは、最新の行政区画情報を反映したCSVファイルを元に生成されています。
- **データ形式**: `{"都道府県名": ["市区町村名1", "市区町村名2", ...]}` というシンプルな構造です。
- **カスタマイズ**: `data/pref-muni.json` を直接編集することで、独自の地域リストや特定のプロジェクトに特化したデータセットに変更することも容易です。

### 活用シーン
- 住所入力フォームのプルダウン連携（都道府県を選択したら市区町村を絞り込むなど）
- 地域情報を扱うアプリケーションのプロトタイプ作成
- Node.js / Express を使用したREST API構築の学習用サンプル

## 🛠️ セットアップ

### 1. クローンとインストール

```bash
git clone https://github.com/s-taku0502/japan-pref-city-api.git
cd japan-pref-city-api
npm install
```

### 2. サーバーの起動

```bash
npm start
# または
node app.js
```

デフォルトでは `http://localhost:3000` でサーバーが立ち上がります。

## 📡 APIエンドポイント

### `GET /api/prefectures`
日本全国の都道府県名一覧を配列で返します。

**レスポンス例:**
```json
["北海道", "青森県", "岩手県", ...]
```

### `GET /api/municipalities?prefecture={都道府県名}`
指定した都道府県に属する市区町村の一覧を返します。

**クエリパラメータ:**
- `prefecture`: 都道府県名（例: `東京都`, `大阪府`, `北海道`）

**レスポンス例:**
```json
["札幌市", "函館市", "小樽市", ...]
```
